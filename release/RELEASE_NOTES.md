## BiViNote v1.2.0

### 新增功能

- 折叠为可拖动圆形图标，点击展开，支持边界限制
- 提示词标签页：AI 笔记整理提示词，根据有无截图自动切换
- 章节/字幕时间戳控制：视频信息页可勾选，默认不带时间戳
- 视频信息每项支持 hover 复制
- 导出目录从 screenshots/ 改为 assets/

### 改进

- 折叠图标位置持久化，展开/折叠不改变用户设定的位置
- 面板和图标默认贴页面右侧
- 边界限制预留 20px 右边距，避免覆盖滚动条
- 字幕滚动定位在第 5 行位置

### 修复

- 折叠图标暗色模式适配
- 折叠图标位置边界 clamp
- 导出下载延迟释放 blob URL
- renderPrompt null guard
- 提示词 {title} 即时更新
- 预览弹窗暗色模式适配
- 多项 Code Review 修复

### 安装方式

1. 下载 bivinote-v1.2.0-chrome.zip
2. 解压到任意目录
3. 打开 Chrome 扩展管理页 (chrome://extensions/)
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」选择解压目录
