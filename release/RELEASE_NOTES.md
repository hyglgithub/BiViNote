## BiViNote v1.3.0

### 新增功能

- 截图浏览界面：基于 Cropper.js 的裁剪、缩放、旋转、翻转
- 截图目录侧栏：始终显示，可快速切换截图
- 工具栏图标+文字：所有按钮同时显示图标和文字
- 图片浏览模式支持拖动

### 改进

- 裁剪框：可调整大小，图片必须覆盖裁剪框
- 旋转/翻转以图片中心为原点
- 切换图片/帧步进无闪烁
- 浏览界面支持白天/黑夜模式
- 背景改为纯色（去除方格背景）

### 修复

- SVG 图标按钮点击无响应
- 裁剪框超出图片范围
- 缩放、旋转、翻转中心点修正
- 帧步进闪烁问题
- 多项 UI 细节修复

### 安装方式

1. 下载 bivinote-v1.3.0-chrome.zip
2. 解压到任意目录
3. 打开 Chrome 扩展管理页 (chrome://extensions/)
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」选择解压目录
