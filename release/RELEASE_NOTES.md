## BiViNote v2.0.0

### 新增功能
- **DeepSeek AI 文档整理** — 接入 DeepSeek 自动整理字幕文档，流式输出思考过程和整理结果
- **文档整理页面** — 开始整理、停止整理、继续询问、下载 Markdown / ZIP
- **提示词管理** — 全屏编辑器，三种提示词独立配置（手动无截图 / 手动有截图 / 自动 DeepSeek）
- **继续询问** — 整理完成后可跳转到 DeepSeek 继续追问

### 改进
- 设置页重构：文档整理方式、提示词管理、恢复默认保留提示词
- CSS 全面适配：字体大小、行高、暗色模式覆盖整个 UI
- 深色模式：文本框滚动条、按钮样式适配
- 视频信息前 6 个选项默认勾选
- 文档整理方式默认为自动

### 修复
- 截图浏览界面帧步进不更新缩略图
- 截图浏览界面工具条和侧边栏不适配字体/行高
- 滚轮事件穿透到背景网页
- 恢复默认设置会重置提示词
- 恢复默认后文档整理页面不刷新

### 安装方式

1. 下载 bivinote-v2.0.0-chrome.zip
2. 解压到任意目录
3. 打开 Chrome 扩展管理页 (chrome://extensions/)
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」选择解压目录
