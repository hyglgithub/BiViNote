使用教程

快速开始

1
下载安装包并解压
从 GitHub 或蓝奏云下载最新版本
2
打开扩展管理页
Chrome: chrome://extensions/  |  Edge: edge://extensions/
3
开启开发者模式，加载扩展
点击「加载已解压的扩展程序」，选择包含 manifest.json 的目录
4
打开 B 站视频页，开始使用
点击浏览器工具栏上的 BiViNote 图标打开面板
